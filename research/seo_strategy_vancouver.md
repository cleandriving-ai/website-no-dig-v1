## 2. Technical SEO Requirements
Using findings from **Awesome SEO**, our site must include:
*   **JSON-LD Schema Markup:** This tells Google exactly what services you offer and where. (I will implement this in the `index.html`).
*   **Core Web Vitals:** Ultra-fast loading speeds. (We will use high-efficiency CSS and optimized images).
*   **Mobile-First Design:** 70% of local service searches happen on phones.

*   **Client Project Highlights:** A "live" feed of recent fixes in specific cities (e.g., "Just fixed a root intrusion in a Langley backyard!").
